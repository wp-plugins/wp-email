» Installation Instructions
  1. Open wp-content/plugins Folder
  2. Put:
    Folder: email
  3. Activate WP-EMail Plugin
  4. Login To WP-Admin, Click E-Mail Tab
  5. Modify your settings accordingly. If the mail server/type settings are not correct, WP-Email will NOT work.
  6. You MAY Need To Re-Generate The Permalink (Options -> Permalinks Options -> Update Permalink Structure)
  7. Refer To Usage For Further Instructions
» Upgrade Instructions
From v2.0x To v2.07
  1. Deactivate WP-EMail Plugin
  2. Open wp-content/plugins Folder
  3. Overwrite:
    Folder: email
  4. Activate WP-EMail Plugin
  5. Go to 'WP-Admin -> EMail -> EMail Options' and restore all the template variables to Default
From v1.0x To v2.07
  1. Deactivate WP-EMail Plugin
  2. Delete these file if exists:
    wp-content/plugins/email.php
    wp-admin/email-options.php
    wp-admin/email-manager.php
    wp-includes/class-phpmailer.php
    wp-includes/class-smtp.php
    wp-includes/functions-wp-email.php
  3. Open wp-content/plugins Folder
  4. Put:
    Folder: email
  5. Activate WP-EMail Plugin
  6. You MAY Need To Re-Generate The Permalink (Options -> Permalinks Options -> Update Permalink Structure)
  7. Refer To Usage For Further Instructions

General Usage
  1. Open wp-content/themes/<YOUR THEME NAME>/index.php

    You may place it in single.php, post.php or page.php also.

  2. Find:
    <?php while (have_posts()) : the_post(); ?>
  3. Add Anywhere Below It:
    <?php if(function_exists('wp_email')) { email_link(); } ?>

    The first value you pass in is the text for emailing post. The second value is the text for emailing page.

    Default: email_link('EMail This Post', 'EMail This Page')

    If you want to use an image/icon instead, replace email_link(); with email_link_image();

    If you want to use a popup window, replace email_link(); with email_popup(); for normal link or email_popup_image(); for image/icon.

  4. Tutorial On How To Integrate wp-email.php With Your Theme:
    http://www.lesterchan.net/wordpress/tutorials/integrating/
EMail Stats (Outside WP Loop)
  • To Display Most E-Mailed Posts
  • Use:
    <?php if (function_exists('get_mostemailed')): ?>
       <?php get_mostemailed(); ?>
    <?php endif; ?>

    The default limit is 10, you can change the limit by doing this <?php get_mostemailed(<LIMIT>); ?>

  • To Display Total E-Mails Sent
  • Use:
    <?php if (function_exists('get_emails')): ?>
       <?php get_emails(); ?>
    <?php endif; ?>
  • To Display Total E-Mails Sent Successfully
  • Use:
    <?php if (function_exists('get_emails_success')): ?>
       <?php get_emails_success(); ?>
    <?php endif; ?>
  • To Display Total E-Mails Sent Unsuccessfully
  • Use:
    <?php if (function_exists('get_emails_failed')): ?>
       <?php get_emails_failed(); ?>
    <?php endif; ?>
